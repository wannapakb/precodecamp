ให้หาว่า input type มีอะไรบ้างแต่ละ type แตกต่างกันอย่างไร

Type ลักษณะ
text ค่าเริ่มต้น. กำหนดฟิลด์ข้อความบรรทัดเดียว
button กำหนดปุ่มคลิกได้
checkbox กำหนดช่องทำเครื่องหมาย
date กำหนดควบคุมวันที่ (ปีเดือนและวัน (ไม่มีเวลา))
file กำหนดข้อมูลไฟล์และเลือกปุ่ม "Browse ... " (สำหรับการอัปโหลดไฟล์)
radio กำหนดปุ่มเพื่อเป็นตัวเลือก
submit กำหนดปุ่มส่ง
password กำหนดฟิลด์รหัสผ่าน
