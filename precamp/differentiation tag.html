ให้หาความเหมือนและความแตกต่างระหว่าง 2 tag ที่กำหนดให้

Tags ความเหมือน ความต่าง
hr & br ไม่ต้องมี tag ปิด hr ใช้สร้างเส้นคั่นระหว่างบรรทัด ,br ใช้ขึ้นบรรทัดใหม่
ul & ol ใช้ในการเรียงลำดับ ul เรียงลำดับแบบใช้สัญลักษณ์,ol เรียงลำดับแบบตัวเลข
select & checkbox & radio button เป็นการสร้างตัวเลือก select แสดงตัวเลือกแบบรายการ เลือกได้ตัวเลือกเดียว,checkbox สามารถเลือกได้หลายตัวเลือก,Radio button จะใช้ในการเลือกแค่ตัวเลือกนั้นตัวเลือกเดียว ไม่สามารถเลือกได้หลายตัวเลือก จะอยู่ในรูปของวงกลมเป็นพื้นที่ว่างๆ
div & span กำหนดรูปแบบหรือลักษณะข้อความ div เป็นองค์ประกอบบล็อก ถูกแสดงทีละองค์ประกอบในแนวตั้ง,span เป็นองค์ประกอบแบบอินไลน์จะถูกแสดงหนึ่งอันตามแนวนอน
script & style กำหนดรูปแบบ css script สามารถเขียน script เช่น javascript เป็น function เพื่อใช้งาน หรือจะเขียนฟังก์ชั่นเก็บเป็นไฟล์ .js ก่อนแล้วค่อยเรียกใช้งาน,style ใช้กําหนดคุณลักษณะ เช่น สี ขนาด font ตำแหน่งการจัดวาง ให้กับ HTML element ต่างๆ
